:running:BGABanner v1.0.0:running:
============
Splash界面导航效果+广告条自动轮播效果

主要功能：Splash界面导航效果、广告条循环自动轮播、自定义指示器位置、手指按下暂停自动轮播

#### [下载Demo](http://bingoshare.u.qiniudn.com/BGABannerDemo.apk)
